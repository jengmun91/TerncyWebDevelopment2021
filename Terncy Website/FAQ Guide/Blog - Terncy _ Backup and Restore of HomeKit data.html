Backup and Restore of HomeKit data

When you need to reset the home center or delete the HomeKit family because of an abnormal gateway, the data in the family may be lost. It’s a huge panic to set device name, allocate room and create scene again. The Home backup function of Terncy app can restore data with one click, and no longer worry about data loss.

If you want to backup and restore Terncy Home Center data, please refer to this article “How to backup and restore Terncy Home Center data”.

Terncy app, backup and restore feature is already supported.

* Note: Terncy APP version needs v2.3.7 or above.

* The backup data of one family can be restored to other families.

* The backup content includes devices, scenes, automations, device names, rooms, etc. in the HomeKit family.

Create a new HomeKit data backup

1.Enter the HomeKit system. In the Terncy app, select the “Me” tab, and select “HomeKit”. Or log in to the home page in Terncy app and click the “HomeKit” button.

2.In Homekit, select the “Me” tab, and select “Home backup”.

3.Click the “➕” button, select the family that needs to be backed up, and click the “next” button.

4.After the backup is completed, add description information and click the “complete” button in the upper right corner to complete the backup.

5.The family data that has been backed up will be displayed in the “Backup and Restore” list.

Restore data from backup

It’s highly recommended to perform Data Store in Terncy system first, and then perform HomeKit data store, which will more effectively ensure the integrity of the data.

The specific operations of HomeKit data restoration are as follows:

1.Tap the “Data restore” button on the data backup details page, choose a homekit family, and tap the “next” button.

2.Tap the “OK” button, waiting for the restore to complete.

3.The restoration was successful.

How to save and use backup data more reliably

Note: The HomeKit family backup data is saved on the mobile phone by default, so the data may be lost when you changes the mobile phone or uninstall Terncy app.

In order to prevent the loss of backup data, you can save the data with share data feature, saving it to Mail, Google Drive, github and other more reliable storages.

Tap the “Save to…” button on the data backup details page, choose the way you want to save.

When you need to use backup data, you can do the following:

1.Download backup file, tap “Open in Other App”, select “Save to Files”, select “Terncy”, and tap “Save”.

2.At this time, the “Home backup” page will display the downloaded backup data.
